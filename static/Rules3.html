Whilst the turn timer is running it is possible for any cylinder to retract it' s move and return to it's original position at the start of the turn. The cylinder may then be moved to a new position as desired.
This option is useful to correct a mistake and can also be used to execute the maneuver "indirect pass through". Please refer to next section.

8) Indirect Pass Through

This maneuver, achievable in one turn, involves two pieces of the same shape. In this example, it is the Blocks Player's turn and for strategic reasons, yellow block is needed the other side of the purple block. The procedure is as follows:
  • Purple block moves to the adjacent black hexagon.
  • Yellow block moves along the edge for two hexagons to land on a blue hexagon.
  • Purple block retracts it's original move and returns to it's start position of a black hexagon

(Optionally, purple block may now move again as in effect it has not yet moved this turn)
